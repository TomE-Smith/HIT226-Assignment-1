Living Style Guides: A Tool for all Web Designers and Developers

What is a Living Style Guide?

A LivingCSS style guide is a CSS (cascading style sheet) document that defines all elements of all the pages that comprise a website (Robertson 2014). These documents are considered living, because the coding inside these documents always apply themselves to any new added pages and/or HTML code changes (Haritonov 2015).

Why is this important?

Living style guides are deemed important for a few key reasons. For instance, these allow web-page coders to check if any part of their page design falters after having code changes (Debenham 2011). Living style guides also allow project newcomers to know what styles to use on any element of the page, as well as allowing the CSS file to remain small and quick to load (Robertson 2014)

How to use Living Style Guides

Some basic tips for creating living style guides:

  • Start out by planning out the general structure of your current project via rough sketches (where the logo sits, where bodys of text lie, where and how the navigation bar works, etc.). (Debanham 2011)

  • Debanham (2011) also writes that after planning the website, people should start defining how the headers, tables and lists, links and bodies of text etc. appear on the page (such as text alignment, colour, font style, etc.)

  • I used this colour palette (black background, grey section backgrounds, yellow text and black hyperlink text) to meet the W3C 1.4.1 guideline (a.k.a. the minimum contrast guideline). This guideline states that the contrast between text and screen (for example) must sit at a minimum ratio of 4.5:1, and 3:1 for large size text(W3C 2008).

    • The following values are the contrast ratios between the various colours in this style guide (courtesy of WebAIM's colour contrast checker)

      • The contrast between the yellow text and the black page background is 17.34:1.

      • The contrast between the yellow text of the footer, list, paragraph elements and the grey section background is 10.88:1.

      • The contrast between the white navigation text and the black navigation bar is 15.46:1.

      • The contrast between the yellow text of the footer, list, paragraph elements and the dark green block representing the active nav tab is 8.37:1.
